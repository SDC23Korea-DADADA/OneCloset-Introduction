## 📱 앱 소개
#### One Closet은 삼성 의류 관리 가전기기 고객들에게 컴퓨터 비전 기술 기반으로 더 편리하고 새로운 서비스를 제공하고 옷장 관리를 돕는 앱 서비스입니다. 의류 이미지에서 분류, 재질, 색상을 추출해 __`관리 코스를 추천하고`__ 옷장에 등록된 의류 이미지를 직접 피팅할 수 있는 __`가상피팅`__ 기능을 제공합니다.
